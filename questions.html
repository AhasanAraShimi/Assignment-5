Question-1: What are the different ways to select an element in the DOM?

There are several way hs to select HTML elements in DOM.They are as follows:

1.document.getElementsByTagName(' '):

In this way, all the elements of the particular HTML tag name can be selected. For example, we can select an

tag by writing .getElementsByTagName('h1'), then all the h1 tags of the HTML file will be selected.

2.document.getElementById(''):

In this way, a particular element can be selected by using the id of that element.this one is the most used method in the case of selecting elements.

3.document.getElementsByClassName(' '):

By a particular class name, all the elements under that class name can be reached.

4.document.querySelectorAll('#id'):

This is another way to reach an element on the basis of id name.In this way all the child inside that particular parent element can be shown in console as childNodes.

5.document.querySelectorAll('.class name'):

Another approach of selecting element on the basis of class name.In this way, all the child elements are shown as childNodes within that particular class name.

6.document.querySelector('.class / #id child name')

when we need to get the first child of a particular parent element we can use this method.
